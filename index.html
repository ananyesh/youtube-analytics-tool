Project Updates

Latest - v1.8 (Real-Time Live Counts)
  • Added: Live real-time subscriber, view, and video counts using Odometer.js (YouTube Studio style).
  • Added: High-frequency polling using estimated subscriber data for a truly "live" experience.
v1.7 (Leaderboard Fix & Cleanup)
  • Fixed: Global Leaderboard now fetches and ranks correctly (resolved API date range issues).
  • Removed: Historical Branding (Wayback Machine) feature has been removed as per user request.
  • Improved: More robust error handling for missing channel stats.
v1.6.1 (Image Fix)
  • Fixed: Search suggestions showing blank/broken profile pictures (API returns "thumbnail" not "thumbnail_url").
  • Fixed: Wayback Machine historical logos failing to load due to overly-strict regex that missed modern YouTube URLs.
  • Added: Multi-level image fallback chain: Wayback proxy → direct URL → YouTube fallback icon.
  • Added: onerror fallback on every single image across the entire app.
v1.6 (Leaderboard Overhaul)
  • Added: Real, curated Top 50 global channels database (MrBeast, T-Series, Cocomelon, etc.) to ensure #1 ranks are accurate.
  • Added: Category Filters! You can now filter the leaderboard by Entertainment, Music, Kids, and Gaming.
  • Fixed: "Most Views" sorting wasn't working due to search API limitations; it now sorts perfectly based on live stats.
v1.5 (Massive Feature Drop)
  • Added: New Compare tab allowing simultaneous graphing of up to 10 channels.
  • Added: New Leaderboard tab with dynamic sorting for Most Subscribers and Most Views.
  • Fixed: Historical logos from Wayback Machine showing up as black circles due to 1x1 tracking pixels.
v1.4.1
  • Added: Version number display next to the logo to easily verify live updates.
  • Fixed: Infinite expanding glitch on the chart caused by unrestricted canvas height.
v1.4
  • Fixed: Broken profile thumbnails in the header and stats section by expanding API key support.
  • Fixed: Graph end-of-line glitches caused by the API returning unformatted hourly data on recent days.
v1.3
  • Added: Real-time search suggestions as you type.
  • Added: Dynamic header logo that changes to the searched channel.
  • Fixed: Wayback Machine image loading issues (using direct image proxy).
  • Added: Navigation tabs for Updates and FAQ.
v1.2

  • Fixed: Chart direction reversed (Oldest first).

Frequently Asked Questions

Why is there no hourly data?

Public statistics are typically archived on a daily basis by major tracking services. Historical hourly data is not publicly archived for the majority of channels.
