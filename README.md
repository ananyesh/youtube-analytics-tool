# YouTube Channel Analytics Tool

A premium, dark-mode web application to track and visualize YouTube channel statistics, growth trends, and historical branding.

## 🚀 Features

- **Real-time Analytics**: Fetch subscribers, views, and video counts via VidIQ API.
- **Growth Visualization**: Interactive charts for daily, weekly, monthly, and yearly growth.
- **Historical Branding**: Scan the Wayback Machine for previous profile pictures and channel names.
- **Data Export**: Download current analytics views as CSV files.
- **Smart URL Parsing**: Supports handles (@name), channel IDs, and legacy URLs.

## 🛠️ Built With

- **HTML5 & CSS3**: Custom dark-mode design with glassmorphism.
- **JavaScript (ES6)**: Logic and API integrations.
- **Chart.js**: For beautiful, responsive charts.
- **Wayback Machine API**: For historical branding retrieval.

## 📦 How to Run

1. Clone or download this repository.
2. Open `index.html` in your browser.
3. *Recommended*: Use a local server (like Live Server in VS Code) for the best performance with historical branding.
